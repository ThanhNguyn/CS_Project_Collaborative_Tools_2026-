## 🚀 Cách sử dụng

### Xem trên GitHub Pages
1. Vào **Settings** → **Pages** → chọn branch `main`, folder `/ (root)`
2. Đợi deploy xong, truy cập link được tạo

### Xem offline
```bash
# Clone repo
git clone https://github.com/your-username/CS_Project_Collaborative_Tools_2026.git

# Mở trực tiếp
start index.html

# Hoặc chạy local server (khuyến khích)
cd CS_Project_Collaborative_Tools_2026
python -m http.server 8080
# Truy cập http://localhost:8080
```
